# Data Structures and Algorithms - C++ Programs

This repository contains a collection of well-documented C++ solutions for various Data Structures and Algorithms (DSA) assignments and exercises. The programs are written using simple syntax to make them easy to learn and understand.

## 📂 Project Structure

The codebase is organized into folders based on specific topics:

- **`Assignment_on_Recursion/`**: Contains 20 C++ programs solving fundamental recursion problems (e.g., Factorial, Fibonacci, Tower of Hanoi, String Reversal).
- **`Recursion_Strings_And_Arrays/`**: Contains 9 C++ programs demonstrating recursive operations on character arrays and strings (e.g., String to Integer conversion, Find First/Last Index, Removing/Replacing characters).
- **`Assignment_on_Hash_Tables/`**: Contains 9 C++ programs demonstrating Hash Table implementations (e.g., Hash Functions, Linear Probing, Separate Chaining, Spell Checker).
- **`Pointers_Structures_Arrays/`**: Contains 10 C++ programs focused on memory management, pointer traversal, and dynamic allocation.
- **`Sorting_Algorithms/`**: Contains 8 C++ programs implementing the most common sorting algorithms (Bubble, Selection, Insertion, Merge, Quick, Heap, Counting, Radix) along with detailed documentation of their Time and Space complexities.
- **`Trees/`**: Contains C++ programs illustrating Tree structures, including Binary Tree Traversals (In-order, Pre-order, Post-order, Level-order), BST operations, and an AVL Self-Balancing Tree implementation.
- **`Graphs/`**: Contains C++ programs demonstrating Graph Representations (Adjacency Matrix/List), Traversals (BFS, DFS), Shortest Path (Dijkstra), and Minimum Spanning Tree (Prim's).
- **`DSA_Practice_2024_2025/`**: Contains a larger project (`Hospital_Management_System.cpp`) which demonstrates the use of multiple Linked Lists in a real-world scenario with an interactive menu.

## 🚀 How to Compile and Run

To run any of the programs in this repository, you will need a C++ compiler installed on your system (such as `g++` on Linux/Mac, or MinGW on Windows).

### 1. Open your terminal

Navigate to the directory of the program you want to run. For example, to run a recursion program:

```bash
cd Assignment_on_Recursion
```

### 2. Compile the code

Use the `g++` command to compile the `.cpp` file into an executable format.

```bash
g++ 1_SumOfArray.cpp -o sum_of_array
```

### 3. Run the executable

After successful compilation, run the output file:

**On Linux / macOS:**
```bash
./sum_of_array
```

**On Windows:**
```cmd
sum_of_array.exe
```
